输入：
运行程序，点击open（快捷键O）

打开.matrix文件
该文件中包含图像文件名，mv矩阵，以及projection矩阵
例如打开的文件为D:\viewpoint\bigBen\201512091557/bigben.matrix
则在其上级目录中应该有相应的原图像存在
在同级目录下的proj文件夹下有相应的mask图像存在

输出：
	cluster Tree
	在同级目录下新建一个以时间命名的文件夹中存储着聚类结果
	
增加了解析器

点击load，(快捷键L)
打开聚类结果的文件夹
例如：E:\ViewPoint\kxm\201511231826\201512151654_19

增加了修改模式
在主窗口或者右边的显示窗口中按Q键可以切换模式
只有两种模式
View模式
	这种模式下用于预览，点击左侧的聚类树，可以查看某一类中的图像
	点击图片可以在文件夹下查看
Choose模式
	这种模式下可以对图像进行删除（Delete），撤销（Ctrl+Z）
	save（S）会将结果保存下来
